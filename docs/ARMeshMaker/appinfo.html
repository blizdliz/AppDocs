ARMeshMaker

App infomation

You can perform a 3D scan and save the space as mesh data.
The mesh data can be exported as a 3D model.
If the device is equipped with a LiDAR sensor, high-definition scanning is possible.

アプリの説明

3Dスキャンを行い、空間をメッシュデータとして保存できます。
メッシュデータを3Dモデルをエクスポートできます。
LiDARセンサーを搭載した端末ですと、高精細なスキャンができます。
