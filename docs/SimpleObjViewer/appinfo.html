3D Splat Viewer

App infomation

This is a viewer application that supports 3D Gaussian Splatting.
You can import multiple 3D models to create your own 3D scene.
Formats such as OBJ and PLY are supported.

アプリの説明

3D Gaussian Splatting に対応したビュワーアプリです。
複数の3Dモデルをインポートして、自分だけの3Dシーンを作成できます。
OBJやPLYなどの形式に対応しています。
